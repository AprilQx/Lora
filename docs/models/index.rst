Overview
--------

Our project utilizes the following key components:

* **Base Model**: Qwen2.5-0.5B-Instruct, a compact yet capable instruction-tuned language model
* **Fine-tuning Method**: Low-Rank Adaptation (LoRA) for parameter-efficient fine-tuning
* **Model Integration**: Custom implementation with FLOP tracking and evaluation metrics
